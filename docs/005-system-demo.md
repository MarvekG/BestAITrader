## 功能概览

天枢智投 `v0.1.0` 覆盖 A 股投研分析、AI 多智能体决策、AI 选股、模拟交易、组合管理、经验复盘、长期记忆、Agent 工具体系、前端控制台和 Docker Compose 一体化部署。
