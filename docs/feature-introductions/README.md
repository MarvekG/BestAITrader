## 2. 使用建议

这些文章默认采用产品推广语气，重点强调价值、场景和差异化。每篇文章都包含 Mermaid 流程图，可直接用于 GitHub、支持 Mermaid 的文档站或产品演示材料。

若用于技术文档或开发者文档，可以搭配根目录 `README.md`、`docs/002-deployment.md` 和后端模块 README 一起使用。
